Embodied Meta Prompting

Project Overview

"Embodied Meta Prompting" is a project that emphasizes the importance of solving embodied tasks step by step. This method leverages Large Language Models (LLMs) to generate experts for each subgoal, aiming to effectively accomplish tasks.

Meta-Prompting Method

Through the meta-prompting method, multiple Vision Language Models (VLMs) can function as specialists for different sub-tasks. For example, an expert specialized in object detection can be generated and coordinated with other specialists working on different subgoals.

Project Goals

The ultimate goal of this project is to establish a new method for efficiently and effectively solving complex embodied tasks. By using LLMs, a meta-approach to task-solving becomes possible, enabling the integration of multiple expert models.
